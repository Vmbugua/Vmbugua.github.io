# Nikkey — Portfolio Website
 
A personal portfolio website showcasing my background, skills, and projects as I work toward a career in data analysis. Built with plain HTML, CSS, and JavaScript — no frameworks or build tools needed.
 
## Pages
 
- **index.html** — Home page with intro and CV download link
- **about.html** — Background and quick facts
- **skills.html** — Technical and soft skills, grouped by category
- **projects.html** — Project showcase (currently placeholders, to be replaced with real work)
- **contact.html** — Contact form and social links
## Supporting files
 
- **style.css** — Shared styles for all pages
- **script.js** — Mobile nav menu, scroll animations, and contact form handling
## Running locally
 
Keep all files in the same folder, then open `index.html` in a browser. No build step required.
 
## Hosting
 
Hosted on GitHub Pages. Changes pushed to `main` go live automatically.
 
## Status
 
Placeholders for the photo, CV link, social links, and projects still need to be replaced with real content.
 
## Built with
 
HTML5, CSS3, Vanilla JavaScript, Montserrat (Google Fonts), Font Awesome
